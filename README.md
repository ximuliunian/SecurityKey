# SecurityKey

SecurityKey 是一个基于 Java 开发的安全工具集，提供了多种密码学相关功能的命令行工具。主要用于处理密钥管理、数据加密/解密、文件签名与验证以及哈希计算等安全操作

## 功能特性

- **密钥管理**
  - 生成密钥对
  - 生成带证书的密钥对

- **数据加密/解密**
  - 非对称加密
  - 非对称解密

- **签名与验证**
  - 文件内容签名
  - 签名验证

- **哈希计算**
  - 计算文件哈希值

## 安装方法

### 前提条件

- Java 17 或更高版本
- Maven 3.6 或更高版本

## 使用示例

### 基本用法

```bash
java -jar SecurityKey.jar [子命令/选项] [选项]
```

### 子命令列表

| 子命令 | 描述 |
|-------|------|
| gen-key | 生成密钥对 |
| gen-key cert | 生成带证书的密钥对 |
| sign | 对文件内容进行签名 |
| encrypt | 对数据进行非对称加密 |
| verify | 对签名进行验证 |
| decrypt | 对数据进行非对称解密 |
| hash         | 计算文件哈希值       |
